# Deutsch Coach

A simple German writing practice app that runs directly in the browser.

## How to use

1. Open `index.html` in a browser.
2. Choose a prompt and level: A1, A2, or B1.
3. Write a few German sentences.
4. Click **Check writing**.

The app gives a score, checks whether your answer matches the selected prompt, judges whether the writing fits the selected German level, shows common beginner corrections, lists strengths, suggests a rewrite, and keeps a short practice history.

The app includes 15 practice prompts covering introductions, routines, past tense, food, family, hobbies, school/work, weather, home/city, shopping, travel, opinions, weekend plans, asking for help, and invitations.

## Files

- `index.html` - page structure
- `style.css` -  design
- `script.js` - scoring and correction logic
